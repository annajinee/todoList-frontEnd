##  Todo List FrontEnd 

​
### 프로젝트 실행 

##### BackEnd Project
https://github.com/annajinee/todoList-backEnd

Backend와 proxy 설정되어 있으므로 위 Backend 프로젝트 Clone - 실행 후, FrontEnd 실행 부탁드립니다.
( proxy.config.json 파일 "target": "http://localhost:8080"로 데이터 요청되도록 설정 되어있습니다.  )

#### 실행 명령어
node.js 없을 시에, node.js 설치 후 실행 
https://nodejs.org/en/
실행 명령어는 아래와 같습니다. 
( todoList-frontEnd/ 폴더에서 명령어 입력 ) 
	
	ng serve --proxy-config ./proxy.config.json
		
#### 실행 주소 


    http://localhost:4200
